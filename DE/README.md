1、计数器级联
设计模为60的计数器（8421BCD码输出）
（1）级联法（分别设计模为10和模为6的计数器，采用同步，异步方式分别级联构成模为60的计数器）
（2）用VHDL设计实现模为60的计数器（8421BCD码输出，带使能控制端和进位输出端）,
并用两个级联测试（类似秒、分计时），用数码管显示结果。
（3）对比级联法与VHDL设计两种方法各自的资源占用情况。

2、点阵/行列式键盘
（1）利用16X16点阵显示自己的名字（逐个显示，例如每秒显示一个汉字）
（2）利用16X16点阵显示自己的名字（滚动显示，上下或者左右方向）
（3）4X4行列式键盘：显示按下键的相应代码（例如0到F）。
（4）在（3）的基础上设计一个简易计算器。


综合设计：
多功能数字钟
（1）基本计时功能：时，分，秒基本计时，12小时制/24小时制可切换（数码管或发光管指示上下午）；
（2）校时功能：
可调整小时，分钟。用两个按键控制，1个按键控制调时模式（调时，调分），1个按键调时。
（3）整点报时功能：
模仿广播电台报时方式，从59分50秒开始，响1秒，停1秒，前5次低音（500Hz），整点高音(1000Hz)。
（4）闹钟功能：
至少可设置一个闹钟（时，分），到时闹钟一直响(按键停止)。闹钟设置方式同调时相同。
（5）将闹钟提示改为播放一段音乐。

交通灯（供学有余力同学）
设计一个十字路口的交通灯控制器，要求如下：
（1）时间固定：东西方向的红灯，绿灯，黄灯亮的时间分别为 50 秒，25 秒，5 秒；南北方向的红灯，绿灯，黄灯亮的时间分别为 30 秒，45 秒，5 秒；
（2）亮灯时间可调节（最大99秒）。
